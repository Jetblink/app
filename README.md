# ctx project

## Introduction

The Ctx service framework is A PHP framework for building modular service application.

## Installation

### Server Requirements

The Ctx service framework has a few basic system requirements:

* PHP >= 5.3

But the Ctx basic module ( include db, cache, config, etc. ) has other system requirements, so it's highly recommendedÂ you make sure your server meets the following requirements:

* PHP >= 7
* PDO PHP Extension
* CURL PHP Extension

### Via composer create-Project

```
composer create-project tree6bee/app=~2.0
```
